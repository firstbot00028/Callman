# 🤖📞 AI Call System (VoIP + AI)

This project is a basic AI-powered call system using VoIP and a backend API.

---

## 🚀 Features
- Incoming call handling via Asterisk
- Basic AI decision logic (urgent / normal)
- Call flow control
- Webhook integration with backend server
- Ready for AI upgrade (OpenAI, TTS, etc.)

---

## 🧠 Tech Stack
- Node.js
- Express.js
- Asterisk (VoIP PBX)
- Render (for hosting backend)
- GitHub (version control)

---

## 📞 VoIP Setup
Configuration files are inside project:

- `sip.conf` → SIP users
- `extensions.conf` → call flow logic
